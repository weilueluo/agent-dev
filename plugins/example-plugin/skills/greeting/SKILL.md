---
name: greeting
description: Generates a friendly greeting. Use when the user asks for a greeting, welcome message, or wants to test that the plugin is working.
version: 1.0.0
---

When asked to greet someone or generate a welcome message:

1. Use a warm, friendly tone
2. Include the person's name if provided via $ARGUMENTS
3. Keep it concise — one or two sentences
4. Optionally add a relevant emoji

If no name is given, greet the user generically.
