# JSON Schemas

This document defines the JSON schemas used by skill-creator.

## evals.json - Defines evals for a skill
## history.json - Tracks version progression in Improve mode
## grading.json - Output from the grader agent
## metrics.json - Output from the executor agent
## timing.json - Wall clock timing for a run
## benchmark.json - Output from Benchmark mode
## comparison.json - Output from blind comparator
## analysis.json - Output from post-hoc analyzer

Each schema has full field descriptions and example JSON.
