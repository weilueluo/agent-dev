# chats

Chat platform integrations for Copilot CLI.

## Skills

| Skill | Description |
|-------|-------------|
| [`whatsapp`](skills/whatsapp/) | Search, send, and manage WhatsApp messages, contacts, and groups via the `wacli` CLI |

## Setup

The whatsapp skill auto-installs `wacli` on first use. You only need to authenticate once:

```
wacli auth
```

This shows a QR code — scan it with your WhatsApp mobile app to link.

After that, the skill handles everything automatically.
